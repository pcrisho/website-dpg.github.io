Nuestra web ofrece calculadora de promedios, recursos académicos, un foro estudiantil y diferentes rutas de aprendizaje para que puedas tomar alguna y puedas seguirla.

Actualmente, todas las funciones son de libre acceso y no requieren registro. En futuras versiones del sistema web se implementará un sistema de cuentas para guardar tu progreso y configuraciones.
